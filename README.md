# **Student Final Score Prediction Using Data Analytics**

## **Brief Overview & Conclusion**

This project analyzes student academic performance using data analytics and machine learning techniques. The dataset includes key factors such as **Study Hours, Attendance, and Previous Scores** to predict the **Final Score** of students.

Through **Exploratory Data Analysis (EDA)**, important patterns and relationships were identified, showing how these factors influence student performance. Regression models were then applied for prediction.

The results showed that **Multiple Linear Regression achieved 92.7% accuracy**, while **Polynomial Regression achieved 92.6% accuracy**, indicating strong predictive performance from both models. Although the difference is minimal, **Multiple Linear Regression performed slightly better due to its simplicity, interpretability, and efficiency**.

👉 **Final Insight:**
**This project demonstrates that data analytics and regression models can effectively predict student performance and support better academic planning and decision-making.**
